# HEARTBEAT.md - 定期检查任务

## ⚠️ 每次心跳先读这个！

**C 盘限制：** 不要向 C 盘写入文件！技能放 G 盘，输出放 F 盘！

---

## 心跳检查清单 (每 6-8 小时)

- [ ] **C 盘空间检查** - 剩余空间 < 10GB 时告警
- [ ] **Temp 目录清理** - 超过 500MB 时清理
- [ ] **记忆文件同步** - 确保 memory/ 目录已备份
- [ ] **敏感文件扫描** - 使用 security-scanner 扫描 API Key/密码泄露
- [ ] **新技能安全检查** - 使用 skillscanner 检查新安装技能

## 每日检查 (每天 09:10)

- [ ] **天气检查** - 四川成都天气预报
- [ ] **日历事件** - 今日/明日安排
- [ ] **每日新闻推送** - 科技新闻/热门话题
- [ ] **系统资源报告** - CPU/内存/磁盘使用情况

## 每周检查 (每周一 17:30)

- [ ] **完整备份** - 执行 backup.ps1 -Full
- [ ] **MEMORY.md 整理** - 回顾上周日志，提炼重要事件
- [ ] **周报告生成** - 生成上周工作总结

## 每月检查 (每月最后一天 17:30)

- [ ] **月度统计报告** - Token 使用/API 配额/任务执行统计
- [ ] **技能使用分析** - 统计最常用的技能
- [ ] **系统健康检查** - 全面检查系统状态
