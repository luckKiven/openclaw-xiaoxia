# Video-Gen-Pro 技能

> **AI 视频生成一站式解决方案**
> 
> 将任意格式的内容（文本、图片、文档）自动转化为完整的视频作品，配套文案和元数据

---

## ⚠️ 依赖要求

### 核心依赖

| 依赖 | 用途 | 必需 | 安装命令 |
|------|------|------|----------|
| **PyYAML** | 工作流配置解析 | ✅ | `pip install pyyaml` |
| **Pillow** | 图片处理 | ✅ | `pip install pillow` |
| **edge-tts** | AI 配音（免费） | ✅ | `pip install edge-tts` |
| **mutagen** | 音频处理 | ⭕ | `pip install mutagen` |
| **requests** | HTTP 请求 | ⭕ | `pip install requests` |

| 依赖 | 用途 | 必需 | 安装命令 |

| **midscene** | 浏览器自动化 | ⭕ | `pip install midscene` |

> 💡 **说明：** midscene 用于调用可灵 AI/即梦 AI 的免费额度（网页自动化），仅在使用 `ai-generate` 命令时需要。

| 工具 | 用途 | 必需 | 安装 |
|------|------|------|------|
| **FFmpeg** | 视频编辑合成 | ✅ | `choco install ffmpeg` 或官网下载 |

## 🎯 功能特性

### 核心功能

| 功能 | 描述 | 状态 |
|------|------|------|
| **多格式输入** | 文本/Markdown/图片/Word/Excel/PDF | ✅ |
| **内容分析** | 主题识别、情感分析、结构提取 | ✅ |
| **脚本生成** | 自动生成视频脚本和旁白 | 🔄 |
| **视觉素材** | AI 生成配图和封面 | 🔄 |
| **配音合成** | ElevenLabs 语音合成 | 🔄 |
| **视频编辑** | FFmpeg 视频合成 | 🔄 |

| **CLIP 验证** | 图片一致性验证 | ✅ |

| 格式 | 处理方式 | 依赖 |
|------|----------|------|
| **纯文本** (.txt) | 直接读取 | - |
| **Markdown** (.md) | 解析结构 | - |
| **图片** (.jpg/.png/.gif/.webp) | 提取元数据 | Pillow |

| **Excel** (.xls/.xlsx/.csv) | 表格解析 | pandas |

## 🚀 使用示例

### 🎬 AI 文生视频（免费额度）

```bash
# 使用可灵 AI 生成（每日 3-5 次免费）
/video-gen-pro ai-generate "电影级特写镜头，2088 年的新上海雨夜，赛博朋克风格" --platform kling

# 使用即梦 AI 生成（新用户送积分）
/video-gen-pro ai-generate "一只可爱的猫咪在阳光下玩耍" --platform jimeng

# 指定时长和画质
/video-gen-pro ai-generate "你的提示词" --platform kling --duration 10 --quality 1080p

# 异步模式（不等待完成）
/video-gen-pro ai-generate "你的提示词" --platform kling --no-wait

# 查看免费额度
/video-gen-pro quota
/video-gen-pro quota --platform kling
/video-gen-pro quota --platform all
```

### 1. 创建主角（可选）

```bash
# 创建新主角
/video-gen-pro protagonist create --name "科技小助手" --images ref1.jpg,ref2.jpg

# 列出所有主角
/video-gen-pro protagonist list

# 删除主角
/video-gen-pro protagonist delete --id char_xxx
```

### 2. 创建项目

```bash
# 从 Markdown 文件创建项目
/video-gen-pro create "AI 科普视频" --input article.md

# 指定主角
/video-gen-pro create "AI 科普视频" --input article.md --protagonist char_001
```

### 3. 生成视频

```bash
# 使用默认模板生成
/video-gen-pro generate --project proj_20260306_001

# 指定模板和质量
/video-gen-pro generate --project proj_20260306_001 --template standard_video_gen --quality high
```

### 4. 查看状态

```bash
# 查看项目状态
/video-gen-pro status --project proj_20260306_001

# 列出所有项目
/video-gen-pro list

# 按状态过滤
/video-gen-pro list --status running
```

### 5. 恢复失败项目

```bash
# 从最后检查点恢复
/video-gen-pro resume --project proj_20260306_001

# 从指定步骤恢复
/video-gen-pro resume --project proj_20260306_001 --from-step generate_images
```

## ⚙️ 配置

### 默认配置 (`config/default_config.yaml`)

```yaml
video_gen_pro:
  default_protagonist: null
  default_template: standard_video_gen
  output_quality: high      # standard/high/ultra
  aspect_ratio: 9:16        # 16:9/9:16/1:1
  auto_publish: false
  
  watermark:
    enabled: true
    text: "Generated by Video-Gen-Pro"
  
  resource_limits:
    max_memory_gb: 8
    max_disk_usage_gb: 50
    max_api_calls_per_minute: 10
  
  cleanup_policy:
    temp_files_ttl_hours: 24
    checkpoint_ttl_days: 7
    cache_ttl_days: 30
```

### 环境变量

```bash
# ElevenLabs API Key（配音必需）
export ELEVENLABS_API_KEY=sk_xxx

# 可选：阿里云 Qwen API（内容生成）
export QWEN_API_KEY=sk_xxx
```

### 自定义工作流

在 `workflows/` 目录创建 YAML 文件：

```yaml
workflow:
  name: quick_gen
  description: "快速生成流程"
  steps:
    - id: parse_input
      handler: "input_adapter.parse_input"
      timeout_seconds: 30
      
    - id: generate_script
      handler: "script_generator.generate"
      depends_on: ["parse_input"]
      
    - id: package_output
      handler: "output_packager.package"
      depends_on: ["generate_script"]
```

### CLIP 一致性验证

- **模型**: clip-vit-base-patch32
- **阈值**: 0.85（可配置）
- **置信度分级**:
  - ≥0.90: 高置信度，自动通过
  - 0.85-0.90: 中置信度，记录日志
  - <0.85: 低置信度，提示用户确认

## ⚠️ 注意事项

1. **API 配额**: ElevenLabs 等 AI 服务有调用限制，注意监控用量
2. **磁盘空间**: 视频生成需要足够的临时存储空间（建议 ≥10GB）
3. **处理时间**: 完整流程可能需要 5-15 分钟，取决于内容复杂度
4. **网络依赖**: 需要联网访问 AI 服务 API
5. **C 盘限制**: 所有输出到 G 盘或 F 盘，不要写入 C 盘
